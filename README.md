# GitHub Desktop

[GitHub Desktop](https://desktop.github.com) is an open-source app designed to help you focus on what matters instead of fighting with Git. This application is built with [Electron](https://www.electronjs.org/), [TypeScript](https://www.typescriptlang.org), and [React](https://reactjs.org/).

### Community Releases

Community-supported package managers:

- **Windows**: 
  - Install using [winget](https://docs.microsoft.com/en-us/windows/package-manager/winget/): `c:\> winget install github-desktop`
  - Or [Chocolatey](https://chocolatey.org/): `c:\> choco install github-desktop`
- **macOS**: 
  - Install using [Homebrew](https://brew.sh/): `$ brew install --cask github`

For Linux, check the [shiftkey/desktop](https://github.com/shiftkey/desktop) fork.

### Facing Issues?

1. **Search Issues**: Check [open issues](https://github.com/desktop/desktop/issues?q=is%3Aopen) and [closed issues](https://github.com/desktop/desktop/issues?q=is%3Aclosed) for a match.
2. **Known Issues**: Review the list of [known issues](https://github.com/desktop/desktop/blob/development/docs/known-issues.md).
3. **Report New Issues**: If no matching issue is found, [open a new issue](https://github.com/desktop/desktop/issues/new/choose) with enough details.

### No Response to Your Issue?

If there’s no response to your issue within a few days, you can politely ping the issue once. Avoid multiple pings.
